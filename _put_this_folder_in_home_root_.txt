Put these folder in the home-root of the user. Default is /home/pi/..


Structure will be:       

/home/pi/Updates/install       (-> update scripts and logic)
/home/pi/Updates/sw/           (-> latest download for updates)
/home/pi/Updates/bu/           (-> backup before replacing sw)

/home/pi/CamController/    (-> the actual sw for the cam)

/home/pi/Services/         (-> system service configurations) 
/home/pi/tools/            (-> script tools and settings) 

/home/pi/WebGui            (-> WebGui for adjusting cam-settings)

/home/pi/shared/images     (folder for sharing images in the network) 
/home/pi/shared/logs       (empty folder for logging)
